# Marketing Copywriting

Conversion-driven marketing copy for landing pages, ads, emails, and sales pages. Bilingual (Hebrew + English). Built for a marketing agency serving small/medium businesses.

## When to use

Trigger phrases:
- "write copy for...", "headline for...", "value prop", "ad copy", "email subject line"
- "כתיבה שיווקית", "טקסט שיווקי", "כתוב לי כותרת", "טקסט לדף נחיתה", "טקסט למודעה", "מייל שיווקי", "סקריפט וידאו"

**Don't use for:** technical documentation, blog SEO content, internal comms.

## Inputs to gather (the brief)

Always collect these BEFORE writing. If user didn't provide, ask in one message:
1. **Business / product** — what they sell, who they are
2. **Target audience** — demographics, psychographics, what they Google at 2am
3. **Pain point** — what hurts the customer now
4. **Promise / outcome** — what changes after the purchase
5. **Differentiation** — why this and not the competitor
6. **Proof** — testimonials, numbers, certifications, case studies
7. **Offer** — price, bonuses, guarantee, urgency
8. **CTA** — what action exactly (book a call, buy now, download)
9. **Tone** — professional / friendly / bold / luxury / playful
10. **Language** — Hebrew, English, or both

## Frameworks to use

Pick by use case:

| Use case | Framework |
|---------|-----------|
| Landing page hero | **PAS** (Problem → Agitate → Solution) or **4U** (Useful, Urgent, Unique, Ultra-specific) |
| Long-form sales page | **AIDA** (Attention, Interest, Desire, Action) |
| Cold email | **PASTOR** or **QUEST** |
| Ad copy (Meta/Google) | **Hook → Promise → Proof → CTA** in 3 lines |
| Headlines | **BAB** (Before-After-Bridge) or "How to [outcome] without [pain]" |
| Email sequences | **Welcome → Story → Value → Pitch → FOMO** |

## Workflow

### 1. Confirm the brief
If brief is incomplete, ask only for missing items in ONE message.

### 2. Draft 3 variations
For headlines/hooks, always produce 3 variations with different angles:
- **Angle A**: emotional / pain-focused
- **Angle B**: outcome / aspirational
- **Angle C**: data / specific number

### 3. Format for the medium
- **Landing page**: hero headline, sub-headline, 3 bullet benefits, social proof, CTA × 2
- **Meta ad**: primary text (125 chars), headline (40 chars), description (30 chars)
- **Google ad**: 15 headlines (30 chars each), 4 descriptions (90 chars each)
- **Email**: subject (40 chars), preview (90 chars), body, CTA button text
- **Sales page**: opening hook, story, problem-agitation, solution reveal, features→benefits, testimonials, offer stack, FAQ, guarantee, CTA

### 4. Hebrew copy rules
- Avoid awkward English transliterations — use natural Hebrew
- Mind RTL formatting — punctuation goes on the correct side
- Read aloud test — does it sound like how the customer actually talks?
- Don't overuse exclamation marks (looks spammy)
- Use second person ("את/אתה") not third person for warmth

### 5. English copy rules
- 8th-grade reading level for B2C, 10th-grade for B2B
- Specific numbers > vague claims ("saves 4 hours/week" > "saves time")
- "You" twice as often as "we"
- Active voice
- Cut adverbs — they weaken claims

### 6. Self-check before delivery
- [ ] Does the headline pass the "so what?" test?
- [ ] Is there one clear CTA per section?
- [ ] Are benefits framed as outcomes, not features?
- [ ] Is there proof to back every bold claim?
- [ ] Did I match the requested tone?

## Deliverables

Provide copy in a clean format:
```
HEADLINE (Option A — emotional):
<text>

HEADLINE (Option B — outcome):
<text>

HEADLINE (Option C — specific):
<text>

SUB-HEADLINE:
<text>

BULLETS (3):
• <benefit>
• <benefit>
• <benefit>

CTA:
<button text>
```

## Anti-patterns

- Generic adjectives ("amazing", "best", "מעולה", "מדהים") with no proof
- "We are passionate about..." — talks about the company, not the customer
- 5-syllable words when 2 will do
- Hyperbole the customer won't believe
- Burying the value prop below the fold
