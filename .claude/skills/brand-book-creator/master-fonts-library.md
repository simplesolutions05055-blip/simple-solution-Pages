# Simple Solutions — Master Fonts Library

Growing library of approved font pairs across agency clients. The visual-system-designer reads this BEFORE recommending fonts, so already-approved pairs get reused for similar industries.

**Goal:** 6-8 master pairs curated over the agency's first 10-15 clients.

- ✅ **Approved master** — proven across 2+ clients, default for matched industries
- 🟡 **Provisional** — used on 1 client, awaiting second use to promote

## How this library grows

1. After every brand book is approved by Gili, the visual-system-designer asks:
   > "האם להוסיף את הזוג ל-master-fonts-library? (כן / לא / ניסיוני)"

2. On "כן" → append a new `### Pair #N — ...` entry, commit to repo.

3. On "ניסיוני" → append but mark as 🧪.

4. On the SECOND client using the same pair → promote 🟡 → ✅ and increment reuse count.

5. Once a pair hits ✅, the matrix reads it as the default candidate #1 for matching industries.

---

## Promotion criteria

| Status | Criteria |
|---|---|
| 🧪 Experimental | First use, no client approval yet, or Gili explicitly wants to test |
| 🟡 Provisional | Used + approved on 1 client |
| ✅ Master | Used + approved on 2+ clients in the same industry cluster |

---

## Anti-patterns

- Don't add a pair just because it looked nice in one mock — wait for client approval.
- Don't promote a pair to ✅ on a single use just because the client loved it — promotion is about reusability.
- Don't recommend pairs marked 🧪 as the default — they're parked, not picked.
