# Brand Book Master Questionnaire

**Fixed and identical for every client.** The order of questions never changes — this is what makes brand-book-creator a repeatable system, not bespoke chaos.

Format: ask questions in batches (3-5 at a time), wait for answers, summarize, continue. Don't dump all 30 at once.

---

## Q0 — Size selector (ALWAYS FIRST)

```
איזה ספר מותג אנחנו בונים?
[1] Mini  — 10-18 עמודים, יסודות + ויזואליה + ערוץ אחד (~3-4 שעות עבודה)
[2] Full  — 22-35 עמודים, כולל Meta+IG+TikTok + WhatsApp + בוטים + מודעות (~6-8 שעות)
```

The choice gates which chapters get built (see `BRAND-BOOK-SKELETON.md`).

## Section A — Existing State (intake of current assets)

Crucial for separating "new from scratch" vs "refresh existing" workflows.

```
1. עסק חדש או קיים?
   (חדש / קיים פחות משנתיים / קיים יותר משנתיים)

2. יש מיתוג כלשהו היום?
   (אין כלום / חלקי — יש לוגו או כמה צבעים / מלא — קיים ספר מותג ישן)

3. אם יש משהו — צרף/תאר:
   - לוגו (SVG/PNG/AI)
   - צבעים מוגדרים (HEX)
   - פונטים בשימוש
   - חומרי פרסום קיימים
   - אתר נוכחי
   - ספר מותג קודם

4. מה הצורך?
   [a] מיתוג חדש לגמרי מאפס
   [b] ריענון מלא לקיים
   [c] השלמת חלקים חסרים בלבד
   [d] רענון ספר מותג ישן
```

## Section B — Business Identity

```
5. שם העסק (משפטי + מסחרי אם שונה)

6. תחום עיסוק / סקטור
   (חשוב — קובע את מטריצת הפונטים. דוגמאות: קוסמטיקה, ייעוץ עסקי, נדל"ן, רפואה, מסעדנות, מוצרי טק, חינוך, אופנה, ספורט, רכב, וכו')

7. סיפור המייסד / סיפור העסק (1-2 פסקאות — איך הכל התחיל, מה הסיפור)

8. שנת הקמה

9. מיקום פיזי (כתובת אם רלוונטי / פעילות ארצית / online only)
```

---

## Section C — Mission / Vision / Values

```
10. למה העסק קיים? מה ה-WHY?  (Mission — משפט אחד עוצמתי)

11. לאן הולכים? איפה רוצים להיות בעוד 5 שנים?  (Vision)

12. 3-5 ערכי ליבה (לדוגמה: שקיפות, חדשנות, מצוינות, חמלה...)

13. הבטחת המותג (one-liner): "אנחנו ה_____ של _____"
```

---

## Section D — Market & Audience

```
14. תיאור הלקוח האידיאלי (פרסונה ראשית):
    - גיל: ___
    - מין: ___
    - הכנסה: ___
    - מיקום בארץ: ___
    - מצב משפחתי: ___
    - סגנון חיים: ___
    - איפה מבלים אונליין: ___

15. פרסונה משנית (אם קיימת)

16. 3 כאבים מרכזיים של הלקוח שאתם פותרים

17. 3 רצונות / חלומות של הלקוח אחרי שהוא צורך מכם

18. 3 מתחרים מובילים — שמותיהם + מה שונה אצלכם מהם
```

---

## Section E — Personality & Archetype

```
19. אם המותג היה אדם — תאר אותו ב-5 תארים
    (לדוגמה: "חכם, ביטחון עצמי, חמים, שובב, מקצוען")

20. 3 מותגים שאתם מעריצים + מה אהוב בכל אחד מהם

21. 3 מותגים שלא רוצים להידמות להם + למה

22. ארכיטיפ ראשי (אחד מ-12 יונגיאניים — אציע אחרי שאקרא את התשובות שלך):
    החכם / הגיבור / האוהב / החוצב-דרך / היוצר / המתמרד / המגן / הקוסם /
    הנאמן / המלאך / המבדח / האדם-הפשוט
```

---

## Section F — Visual Preferences

```
23. גוונים שמושכים אותך / גוונים שאתה לא אוהב

24. סגנון ויזואלי כללי (בחר את הכי קרוב):
    [a] מינימליסטי-נקי (Apple, Linear)
    [b] מתחכם-אדיטוריאלי (Aesop, Cereal Magazine)
    [c] חם-אנושי (Mailchimp, Duolingo)
    [d] טכני-מדויק (Stripe, Vercel)
    [e] יוקרתי-קלאסי (Chanel, Hermès)
    [f] שובב-צבעוני (Notion, Figma)
    [g] רטרו / נוסטלגי
    [h] הוליסטי-אורגני (kombucha brands, yoga studios)

25. השראות חזותיות — קישורים לאתרים, חשבונות אינסטגרם, Pinterest boards

26. הרגש שצריך לעורר בלקוח כשהוא רואה את המותג בפעם הראשונה
    (לדוגמה: "אני סוף סוף בידיים טובות", "זה מקצועי באמת", "אני רוצה להיות חלק מזה")
```

## Section G — Practical / Operational

```
27. שפות פעילות:
    [a] עברית בלבד
    [b] עברית + אנגלית (RTL + LTR)
    [c] אנגלית בלבד

28. תקציב לרישוי פונטים:
    [a] חינמי בלבד — Google Fonts
    [b] ₪500-1500 — אפשרי FontBit / HaTipusiya / רישיון ענן בודד
    [c] ₪1500+ — אפשרות לפונטים מקצועיים מלאים

29. ערוצי פרסום עיקריים (סמן את הרלוונטיים):
    [ ] Instagram
    [ ] Facebook
    [ ] TikTok
    [ ] LinkedIn
    [ ] YouTube
    [ ] WhatsApp Business
    [ ] Print (פליירים, שלטי חוצות)
    [ ] TV / רדיו
    [ ] אתר אינטרנט / SEO
    [ ] Email marketing

30. מועד יעד להשקה / סיום ספר המותג
```

## Post-questionnaire summary template

```

גודל ספר: Mini / Full
תחום: ___
שם: ___
שפות: ___
תקציב פונטים: ___

קהל יעד עיקרי: [גיל, מין, אזור, סגנון חיים]
5 כאבים/רצונות: ___

אישיות (5 תארים): ___
ארכיטיפ נטוי: ___ (יאומת על ידי הסטרטג)

סגנון ויזואלי מועדף: ___
רגש מטרה: ___

ערוצים: ___

```
