# Israeli Boutique References

Real Israeli boutique designer/agency sites that set the quality bar this skill must hit. The `lp-design-dna-blender` MUST consult this file (in addition to the international `design-references-index.md`) for every Israeli client project.

These are SENIOR-MASTER reference sites — the ones Gili measures himself against. The goal is for the agency's output to feel like it belongs alongside these.

---

## Reference 1 — s-pixel.co.il

**Genre:** Israeli web design studio.
**Why it matters:** Hits the editorial-tech sweet spot. Generous whitespace, big restrained typography, careful motion.

**What to borrow:**
- Generous section spacing — sections breathe at 120-160px vertical padding
- Restrained color — one dominant ink color + one bold accent + ivory/paper ground
- Real photography (their work showcase) — never stock
- Section transitions with subtle parallax — not flashy
- Typography hierarchy with clear weight + size jumps (no muddle)

**What NOT to borrow:**
- Direct typography pairing (use a different Hebrew display font to differentiate)
- Specific color palette (find a different accent — they're branded around theirs)

**Use when:** client is a service business that wants to feel premium-but-grounded (consultants, agencies, photographers, architects).

## Reference 2 — bluebee.co.il

**Genre:** Israeli marketing/business consultancy.
**Why it matters:** Strong page structure + conversational Hebrew copy style. The writing in particular is a model — direct, opinionated, not buzzwordy.

**What to borrow:**
- Page structure rhythm — clear sections with single jobs, no over-packed cards
- Hebrew copywriting tone — first-person, opinionated, takes a stance
- CTAs written as full sentences (not "התחל") — committed action
- Lead-capture form integrated naturally, not floating overlay

**What NOT to borrow:**
- Specific palette
- Specific section ordering (each client gets their own ordering)

**Use when:** client offers consulting, coaching, B2B services where TONE matters more than visual fireworks.

---

## Reference 3 — alexgorbachov.co.il/newpage/

**Genre:** Israeli personal brand / consultant landing.
**Why it matters:** Strong founder-led design. Real photography of the founder, integrated into the layout — not a "founder card with crossed arms in blazer".

**What to borrow:**
- Founder-led photography woven through the design (multiple shots, different contexts)
- Layout where the founder's voice/face IS the design language
- Direct headline + sub-head rhythm — no fluff
- Testimonials presented as long-form quotes, not 15-word slogans

**What NOT to borrow:**
- The specific photographic style (each founder needs their own)

**Use when:** client is a personal-brand consultant / coach / expert whose face IS the offer.

---

## Reference 4 — webiztlv.co.il

**Genre:** Israeli web design agency (Tel Aviv).
**Why it matters:** Asymmetric layouts done well. Mixed grid widths, deliberate negative space, sections that don't all look the same.

**What to borrow:**
- Asymmetric grids — 5/7, 4/8, full-bleed, narrow-centered all in one page
- Section variety — no two sections feel templated the same way
- Scroll-triggered subtle motion — present but never showy
- Real client work as the design centerpiece, not decoration

**What NOT to borrow:**
- Specific aesthetic moves (their palette/typography is their brand)

**Use when:** client is in a creative or design-adjacent field (photographer, brand designer, art director, architect).

## Selection logic for the design blender

When designing for an Israeli client:

1. **Always pick at least ONE reference from this file** as a quality anchor.
2. **Combine with 1-2 international references** from `design-references-index.md` for differentiation.
3. **NEVER clone the chosen reference.** Borrow specific lessons (typography rhythm, layout move, copy tone) — not the whole feel.


## What these references have in common (the bar)

All four share these traits — and the skill's output MUST also share them:

1. **Visual restraint.** Sections breathe. No "every-section-gets-decoration" pattern.
2. **Real content.** Real photography, real client work, real testimonials. No placeholder feel.
3. **Distinctive typography.** Each picks a Hebrew font that's NOT Heebo-by-default. Custom hierarchy.
4. **Asymmetric grids.** No 3-column-cards-everywhere.
5. **Subtle but deliberate motion.** Not 20 micro-animations — 2-3 well-chosen ones.
6. **Strong opinion in copy.** First-person, direct, opinionated. Not "we help X do Y".
7. **One signature moment.** A custom cursor, a parallax reveal, a typographic effect — one thing that's THEIRS.
8. **NO em-dashes in body copy.** The Hebrew reads naturally; AI-template dashes are absent.
9. **NO navy + gold "luxury template".** They use distinctive palettes that fit each studio's brand.
10. **Generous whitespace.** Sections at 100-160px vertical padding, not 60-80px crammed.
