# Design References Index

71 world-class brand references live in `.claude/references/awesome-design-md/design-md/`. The `lp-design-dna-blender` agent picks 2-3 of them per client and blends them into a unique visual DNA — never a clone.

**Rule:** Read each chosen reference's markdown BEFORE writing tokens. Extract its rules — type scale, motion easing, grid, color logic, photography direction — then blend.

---

## How to read this index

Each reference is tagged with:
- **Mood** — luxury / editorial / tech / consumer / industrial / organic / clinical
- **Strength** — what this brand does better than anyone (the lesson)
- **Best blend for** — LP types this reference fits well in

## Blending rules

1. **Never pick just one** — always 2-3 references, blended.
2. **Identify what each contributes** — typography from one, motion from another, color logic from a third.
3. **Document the blend** — `DESIGN-LANGUAGE.md` lists each reference + the specific element borrowed.
4. **Never copy verbatim** — if a section feels like it could BE the reference brand, it's wrong. Blend, don't clone.
5. **One client's blend never repeats for another** — even in the same industry, the second client gets a different blend.

## Worked examples

### Lawyer for tech founders, Herzliya Pituach
**Blend:** `stripe` (clarity, information density) + `linear.app` (monochrome restraint, micro-grid) + 15% `apple` (oversized weight, hairline dividers).
**Result:** monochrome `#0A0A0A` on `#FAFAFA`, electric accent `#3B5BFF`, mono-display headline, Hebrew body in Heebo 400/500, 96px+ section spacing, subtle 1px micro-grid in hero, abstract geometric SVG only (no people photos).

### Vegan restaurant, פלורנטין
**Blend:** `apple`-editorial-via-`airbnb`-warmth + 20% `webflow` playfulness.
**Result:** `#F4EFE4` cream + `#2A3B1F` deep moss + `#C75A2C` terracotta accent, Suez One display, Assistant body, hand-drawn SVG vegetable line illustrations, asymmetric stacked layouts, warm tungsten food photography.

### B2B SaaS by ex-8200 founders
**Blend:** `linear.app` + `vercel` + 10% `cursor` energy.
**Result:** deep `#0B0B0F` near-black, `#E6E6F0` off-white, lime `#B8FF6B` accent, Inter Display ExtraBold tracked tight, scroll-jacking product demo, glassmorphism panels at 4% opacity, animated noise grain at 3% over hero.
